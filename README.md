# >_ DevCore OS

**The developer-first Linux OS. Built for builders.**

DevCore OS is a custom Linux distribution based on Ubuntu 24.04 LTS, 
pre-loaded with everything a developer needs — right out of the box. 
No setup. No configuration. Just boot and build.

---

## 🚀 Features

- **Zero setup** — Boot and start coding immediately
- **Pre-installed dev tools** — Git, Node.js, Python, VS Code, Zsh and more
- **AI-ready** — Ollama pre-installed on Pro edition
- **Lightweight** — Lite edition runs on 2GB RAM
- **Beautiful** — Custom dark theme, dock, and DevCore branding

---

## 📦 Editions

| | DevCore Lite | DevCore Pro |
|---|---|---|
| RAM | 2GB minimum | 8GB minimum |
| Desktop | XFCE | GNOME |
| Tools | Core dev stack | Full stack + AI + Security |
